## в—Ҳ What is LoanMind?

> **LoanMind** is a Java-powered financial planning tool that helps users simulate, analyze, and understand real-world loan scenarios вҖ” mortgages, student loans, personal loans, and more. Built as an IB Computer Science SL project, it combines rigorous financial mathematics with modern software architecture.

The core idea is simple: **demystify loans**. Interest rates, repayment schedules, and total costs can be confusing. LoanMind makes them visual, interactive, and clear.

### рҹҸҰ Loan Calculation Engine

Calculate with precision:

- вңҰ **Monthly repayment amount** using standard amortization formula
- вңҰ **Total payment** over the full loan lifetime
- вңҰ **Total interest paid** vs principal breakdown
- вңҰ **Comparison** across different loan conditions
- вңҰ **Remaining balance** tracked for every period

**рҹ”· Object-Oriented Programming**
The system is divided into dedicated classes вҖ” `Loan`, `Payment`, `LoanCalculator` вҖ” each with a clear single responsibility. Inheritance allows different loan types to share core logic while overriding specific behaviors.

**рҹ”· Arrays & Collections**
Repayment schedules are stored as ordered collections, enabling indexed access to any month's payment data and full iteration over the entire plan.

**рҹ”· Loops & Iteration**
The amortization engine loops through every period, calculating interest, reducing principal, and updating balance until the loan is fully repaid.

**рҹ”· Input Validation**
Guards against negative values, empty inputs, zero-term loans, and unrealistic financial data вҖ” improving both reliability and user experience.

## в—Ҳ The Math Behind It

<br>

LoanMind uses the standard **amortization formula** to calculate fixed monthly payments:

<br>

```
         P Г— r Г— (1 + r)вҒҝ
  M  =  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
           (1 + r)вҒҝ вҲ’ 1

  Where:
    M  =  Monthly payment
    P  =  Principal (loan amount)
    r  =  Monthly interest rate  (APR Г· 12)
    n  =  Total number of payments  (years Г— 12)
```

<br>

Each month, interest is recalculated on the **remaining balance**, so the interest portion shrinks and the principal portion grows вҖ” this is the amortization effect.
